# Recursos de Android para MultiCajas

Este directorio contiene recursos personalizados para la aplicación Android.

## 📁 Archivos Incluidos

- `strings.xml` - Cadenas de texto localizadas
- `colors.xml` - Definición de colores
- `ic_launcher.xml` - Icono de launcher (vector)

## 🎨 Personalización

### Cambiar Colores
Editar `colors.xml` y modificar los valores hexadecimales.

### Cambiar Nombre de la App
Editar `strings.xml` y modificar `<string name="app_name">`.

### Cambiar Icono
Reemplazar `ic_launcher.xml` con tu propio vector o usar Android Asset Studio:
https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html

## 📱 Densidades de Pantalla

Para iconos PNG, colocar en:
- `res/mipmap-mdpi/` (160dpi)
- `res/mipmap-hdpi/` (240dpi)
- `res/mipmap-xhdpi/` (320dpi)
- `res/mipmap-xxhdpi/` (480dpi)
- `res/mipmap-xxxhdpi/` (640dpi)
